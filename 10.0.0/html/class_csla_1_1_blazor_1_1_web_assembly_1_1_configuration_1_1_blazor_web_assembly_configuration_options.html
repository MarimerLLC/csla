bool Csla.Blazor.WebAssembly.Configuration.BlazorWebAssemblyConfigurationOptions.SyncContextWithServer = false
getset

Gets or sets a value indicating whether the LocalContext and ClientContext values on ApplicationContext should be synchronized with a Blazor web server host.

If this value is true, the Blazor web server host must provide a state controller to allow the wasm client to get and send the state from/to the server as necessary.
