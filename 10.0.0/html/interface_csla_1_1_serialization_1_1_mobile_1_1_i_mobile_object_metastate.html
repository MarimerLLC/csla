Detailed Description

Interface for types that can serialize their metastate (field values) to and from a byte array representation.

This interface provides a lightweight serialization mechanism that captures the same field values as OnGetState/OnSetState without including child object references. It is intended for transitory data serialization scenarios, not long-term storage.
